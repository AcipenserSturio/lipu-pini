sona Linku (the dataset)

The dataset is a github repository which has all data we track, and is where you go to add new data. Everyone may view the data, but direct contributions are only available to maintainers or via pull requests.

Changes to the source language data are automatically synced to Crowdin, and changes to Crowdin translations are automatically synced to the dataset. Subsequently, all data in the dataset is made available via the API hosted alongside the website.

The Linku github organization is owned by kala Asi and jan Kekan San, and changes are discussed in kulupu Linku.

If you are looking for the old dataset called jasima, the data can be found here and the repo can be found here. Also see nimi Linku for information about the previous Google sheet.
